Download the following models and place them inside this folder:

* 5 and 68 facial landmark predictors (consult https://github.com/davisking/dlib-models#shape_predictor_5_face_landmarksdatbz2 for download links)
* ResNet face recognition model (https://github.com/davisking/dlib-models#dlib_face_recognition_resnet_model_v1datbz2)
* OpenFace model (https://storage.cmusatyalab.org/openface-models/nn4.v2.t7)


By default, this folder is automatically copied to the target directory when project building completes. It can be disabled by setting the `COPY_MODELS` option off.
